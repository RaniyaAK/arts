    Digital Art Commission Management System

    Paletra is a role-based web application that connects artists and clients for managing custom digital art commissions. The system focuses on commission requests, progress tracking, advance payments, and structured delivery.

    Get Started

    System Roles

    Artist

    Artists upload sample artworks to showcase their style, receive commission requests from clients, request advance payments, and update commission progress until delivery.

    Client

    Clients browse artist profiles and sample artworks, submit commission requests with requirements, pay advance amounts, track progress, and receive completed digital artwork.

    Commission Workflow

    Step 1

    Client selects an artist and submits a commission request with description and reference image.

    Step 2

    Artist reviews the request and accepts or rejects it. Artist can request an advance payment.

    Step 3

    Client pays advance via PayPal. Artist starts work and updates commission status.

    Step 4

    Artwork is completed, shipped digitally, and marked as delivered.

    About the Project

    This project is developed using Django framework and follows a role-based architecture. The application demonstrates authentication, profile management, commission workflows, payment integration, notifications, and dashboard-based user interaction.
